# Installation
- just clone this repo
```
$ git clone https://github.com/ADIKOLORD/flask1
```
- open repo directiory 
```
$ cd flask1
```

- create vertualenv && activate vertualenv
```
$ python3 -m venv venv
$ . ./venv/bin/activate
```
- install dependencies
```
$ pip install -r related.txt
```
- run project
```
$ export FLASK_APP=flask1
$ flask run
```
